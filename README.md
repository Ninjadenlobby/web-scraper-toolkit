# web-scraper-toolkit

A simple Python web scraper for extracting data from websites.

## Features

- Scrape page titles and meta descriptions
- Extract all links from a page
- Export results to JSON
- Rate limiting to be polite

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python scraper.py https://example.com
python scraper.py https://example.com --links --output results.json
```
